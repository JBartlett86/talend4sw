What is talend4sw ?

Talend4SW is a project that aims to offer Talend components to produce and consume RDF from the Semantic web.

The available components belong to one of the three following category :

Virtuoso Triplestore components

The goal of these components is to make operations on Virtuoso triplestore easily using Talend. These components will allow the user to connect to a Virtuoso triplestore, submit sparql query, insert data, clear graph...

Jena components

The goal of these components is to easily create and make operations on Jena model using Talend. These components will allow the user to create a Jena model, to add statements and to query it.

Semantic data components

Those components aimed to make Talend more user friendly to transform RDF data.

Virtuoso Triplestore components

The goal of these components is to make operations on Virtuoso triplestore easily using Talend. These components will allow the user to connect to a Virtuoso triplestore, submit sparql query, insert data, clear graph...

tVirtuosoConnection

The user uses this component to connect to a Virtuoso triplestore by informing the host, the port, the username and the password

tVirtuosoSelect

Submit a sparql select query to a tVirtuosoConnection.

tVirtuosoConstruct

Submit a sparql construct query to a tVirtuosoConnection

tVirtuosoInsert

Insert statements in a virtuoso triplestore (defined by a tVirtuosoConnection)

tVirtuosoClearGraph

Clear a graph in a Virtuoso triplestore (defined by a tVirtuosoConnection)

Jena components

The goal of these components is to easily create and make operations on Jena model using Talend. These components will allow the user to create a Jena model, to add statements and to query it.

tJenaCreateModel

Create an empty Jena model (a set of RDF statements).

tJenaReadFile

Fill a model (created using tJenaCreateModel) with statements from a file (RDF/XML, n-triple or turtle format)

tJenaReadRow

Fill a model with statements from an incoming row (RDF/XML, n-triple or turtle format)

tJenaSelectQuery

Submit a sparql select query to the Jena model.

tJenaConstructQuery

Submit a sparql construct query to the Jena model.

tJenaDropModel

Clear a model from any statements.

Semantic data components

Those components aimed to make Talend more user friendly to transform RDF data.

tRDF2RDF

Convert data from a RDF format to another RDF format. The format available are RDF/XML, n-triple and Turtle.

tTriple2Ntriple

Convert a 4 columns row (subject, predicate, object, type) into a well formed n-triple. It also replace common prefixes and the user can add prefixes to be replaced during the transformation.
